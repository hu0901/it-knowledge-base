# IP 地址冲突

## 故障现象

系统提示"IP 地址冲突"，网络连接不稳定。

## 原因

两台设备使用了相同的 IP 地址。

## 解决方案

### 自动获取 IP

设置 → 网络和 Internet → WLAN/以太网 → 硬件属性 → IP分配 → 编辑 → 自动(DHCP)

### 手动释放和更新

```
# 释放当前IP
ipconfig /release

# 获取新IP
ipconfig /renew
```

### 路由器端设置

1. 登录路由器管理界面
2. 查看 DHCP 客户端列表
3. 检查静态 IP 分配
4. 调整 DHCP 地址池范围

## 预防措施

- 使用 DHCP 自动分配
- 静态 IP 做好记录
- 定期检查网络设备
