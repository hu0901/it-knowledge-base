# VPN 连接问题

## 故障现象

VPN 无法连接、连接后无法访问内网资源。

## 排查步骤

### 1. 检查 VPN 配置

- 确认服务器地址正确
- 检查用户名密码
- 确认 VPN 类型（PPTP/L2TP/SSTP/IKEv2）

### 2. 检查网络连接

测试 VPN 服务器连通性，ping 服务器地址。

### 3. 防火墙检查

- 关闭 Windows 防火墙测试
- 检查杀毒软件拦截
- 确认路由器允许 VPN 协议

### 4. 常见错误代码

| 错误代码 | 原因 | 解决方案 |
|---------|------|---------|
| 800 | 无法建立连接 | 检查服务器地址和端口 |
| 809 | 无法完成连接 | 检查防火墙设置 |
| 691 | 用户名或密码错误 | 重新输入凭据 |
| 720 | 协议配置错误 | 重新配置 VPN |

### 5. 重置 VPN

设置 → 网络和 Internet → VPN → 删除并重新添加
