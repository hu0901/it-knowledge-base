# 网络速度慢

## 故障现象

网页加载慢、视频卡顿、下载速度慢。

## 排查步骤

### 1. 测速

访问 speedtest.cn 或 speedtest.net 测试实际带宽。

### 2. 检查带宽占用

按 Ctrl+Shift+Esc 打开任务管理器，查看网络使用情况。

### 3. 检查路由器

- 重启路由器
- 检查连接设备数量
- 查看是否有设备大量下载

### 4. WiFi 优化

- 切换 WiFi 信道（避开拥挤信道）
- 5GHz 频段优先
- 靠近路由器测试

### 5. DNS 优化

更换为公共 DNS：
- 阿里 DNS: 223.5.5.5, 223.6.6.6
- 腾讯 DNS: 119.29.29.29
- 114 DNS: 114.114.114.114

## 常见原因

- 带宽被其他设备/程序占用
- WiFi 信号干扰
- DNS 解析慢
- 路由器性能不足
- 运营商网络问题
