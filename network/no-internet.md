# 无法上网

## 故障现象

浏览器无法打开网页，显示"无法连接到互联网"。

## 快速排查流程

```
1. 检查网线/WiFi连接
2. 重启路由器和电脑
3. 检查IP配置
4. 重置网络设置
```

## 详细排查

### 1. 检查物理连接

- [ ] 网线插紧，指示灯正常
- [ ] WiFi 已连接且信号良好
- [ ] 路由器/光猫正常运行

### 2. 网络诊断命令

```
# 检查IP配置
ipconfig /all

# 测试网关连通
ping 192.168.1.1

# 测试DNS解析
ping 8.8.8.8
ping www.baidu.com

# 刷新DNS缓存
ipconfig /flushdns

# 重置网络
netsh winsock reset
netsh int ip reset
```

### 3. 检查代理设置

设置 → 网络和 Internet → 代理 → 关闭代理服务器

### 4. 重置网络适配器

设置 → 网络和 Internet → 高级网络设置 → 网络重置

## 常见原因

| 原因 | 检查方法 | 解决方案 |
|------|---------|---------|
| 路由器故障 | 其他设备也无法上网 | 重启路由器 |
| DNS问题 | 能ping通IP但打不开网页 | 更换DNS |
| IP冲突 | ipconfig显示169.254.x.x | 释放并更新IP |
| 代理设置 | 检查代理配置 | 关闭代理 |
| 网卡驱动 | 设备管理器检查 | 更新驱动 |
