# 打印机连接问题

## 故障现象

打印机无法打印、脱机状态或无法识别。

## 排查步骤

### USB 打印机

1. **检查物理连接**
   - 更换 USB 端口
   - 更换 USB 线缆
   - 检查打印机电源

2. **检查打印机状态**
   - 确认打印机已开机
   - 检查纸张和墨盒
   - 查看打印机屏幕错误信息

3. **重新安装驱动**
   - 删除原有打印机
   - 从官网下载最新驱动
   - 重新安装

### 网络打印机

1. **检查网络连接**
   - 确认打印机 IP 地址
   - Ping 打印机 IP 测试连通性
   - 检查网络线缆连接

2. **重新添加打印机**
   设置 → 设备 → 打印机和扫描仪 → 添加打印机

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 打印队列卡住 | 清空打印队列，重启 Print Spooler 服务 |
| 驱动不兼容 | 下载对应系统版本的驱动 |
| 脱机状态 | 取消"脱机使用打印机"勾选 |
