# 显示器无信号/黑屏

## 故障现象

电脑主机运行正常，但显示器显示"无信号"或黑屏。

## 排查步骤

### 1. 检查视频线连接

- [ ] 确认视频线两端插紧
- [ ] 尝试更换视频线
- [ ] 尝试更换视频接口（如 HDMI 换 DP）

### 2. 检查显示器设置

- [ ] 确认显示器电源开启
- [ ] 检查显示器输入源设置
- [ ] 尝试显示器菜单按钮

### 3. 显卡检查

- [ ] 重新插拔显卡（独立显卡）
- [ ] 检查显卡供电线
- [ ] 尝试集成显卡输出

## 常见原因

- 视频线松动或损坏
- 显卡驱动问题
- 分辨率设置过高
- 显卡硬件故障
