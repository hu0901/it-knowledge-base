# 电脑无法开机

## 故障现象

按下电源按钮后，电脑没有任何反应，或风扇转但屏幕不亮。

## 排查步骤

### 1. 检查电源连接

- [ ] 确认电源线插紧
- [ ] 尝试更换电源插座
- [ ] 检查电源适配器指示灯（笔记本）

### 2. 检查显示器（台式机）

- [ ] 确认显示器电源开启
- [ ] 检查视频线连接（HDMI/DP/VGA）
- [ ] 尝试切换显示器输入源

### 3. 最小化启动测试

断开所有外设，只保留：
- 电源线
- 显示器
- 键盘

### 4. 清除静电

1. 拔掉电源线
2. 按住电源按钮 30 秒
3. 重新连接电源尝试开机

### 5. 检查内存条

1. 关机断电
2. 打开机箱
3. 重新插拔内存条
4. 如有多个内存条，逐一测试

## 常见原因

| 原因 | 概率 | 解决方案 |
|------|------|---------|
| 电源故障 | 30% | 更换电源测试 |
| 内存接触不良 | 25% | 重新插拔内存 |
| 静电积累 | 20% | 清除静电 |
| 主板故障 | 15% | 送修检测 |
| 其他硬件 | 10% | 逐一排查 |
