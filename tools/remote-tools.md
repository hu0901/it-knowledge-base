# 远程协助工具

## TeamViewer

### 特点
- 跨平台支持（Windows/Mac/Linux/手机）
- 无需安装，可运行便携版
- 支持文件传输

### 使用方法
1. 双方下载并运行 TeamViewer
2. 被控方提供 ID 和密码
3. 控制方输入 ID 连接

## AnyDesk

### 特点
- 轻量级，启动快
- 低延迟
- 免费版功能完整

### 使用方法
1. 运行 AnyDesk
2. 被控方提供 AnyDesk 地址
3. 控制方输入地址连接

## Windows 远程桌面

### 启用远程桌面
设置 → 系统 → 远程桌面 → 启用远程桌面

### 连接方法
1. 按 Win+R → 输入 mstsc
2. 输入目标计算机 IP 或计算机名
3. 输入用户名和密码

### 注意事项
- 需要目标电脑开启远程桌面功能
- 需要在同一网络或配置端口转发
- 家庭版 Windows 不支持被远程控制

## 向日葵

国产远程控制软件，适合国内网络环境。

## 选择建议

| 场景 | 推荐工具 |
|------|---------|
| 临时协助 | AnyDesk、TeamViewer |
| 长期管理 | Windows 远程桌面 |
| 国内环境 | 向日葵 |
| 跨平台 | TeamViewer |
