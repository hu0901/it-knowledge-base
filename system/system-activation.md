# 系统激活问题

## 常见激活方式

### 1. 数字许可证激活

适用于 Windows 10/11 从已激活的 Windows 7/8.1 升级而来。

### 2. 产品密钥激活

设置 → 系统 → 激活 → 更改产品密钥

### 3. KMS 激活（企业环境）

```
slmgr /skms kms.server.com
slmgr /ato
```

## 常见激活错误

| 错误代码 | 原因 | 解决方案 |
|---------|------|---------|
| 0xC004C003 | 密钥被阻止 | 联系微软支持 |
| 0xC004F034 | 密钥无效 | 检查密钥输入 |
| 0xC004F050 | 密钥已被使用 | 使用电话激活 |
| 0x803F7001 | 未找到数字许可证 | 登录微软账户 |

## 电话激活步骤

1. 按 Win+R → 输入 slui 4
2. 选择国家/地区
3. 拨打显示的电话号码
4. 按语音提示输入安装 ID
5. 输入获得的确认 ID

## 检查激活状态

按 Win+R → 输入 slmgr /xpr 查看激活状态。
