# 软件安装失败

## 常见原因

| 原因 | 表现 | 解决方案 |
|------|------|---------|
| 权限不足 | 提示需要管理员权限 | 右键以管理员身份运行 |
| 磁盘空间不足 | 安装程序报错 | 清理磁盘空间 |
| 杀毒软件拦截 | 安装过程被阻止 | 暂时关闭杀毒软件 |
| .NET Framework 缺失 | 提示缺少组件 | 安装对应版本的 .NET |
| Visual C++ 运行库缺失 | 运行时报错 | 安装 VC++ 运行库 |
| 安装目录不存在 | 无法创建目标文件夹 | 手动创建目录 |

## 排查步骤

### 1. 检查权限

- 右键安装程序 → 以管理员身份运行
- 关闭 User Account Control (UAC) 临时测试

### 2. 检查依赖组件

常用依赖组件：
- .NET Framework 3.5 / 4.8
- Visual C++ 2005-2022 运行库
- DirectX 9.0c

### 3. 干净启动排查

1. 按 Win+R → 输入 msconfig
2. 选择有选择的启动
3. 取消加载启动项
4. 重启后重试安装

## 常见错误代码

- 0x80070005: 权限不足
- 0x8007000D: 文件损坏，尝试重新下载
- 0x80070643: 安装缓存损坏，运行清理工具
