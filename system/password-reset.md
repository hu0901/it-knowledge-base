# 密码重置

## 本地账户密码重置

### 方法一：使用密码重置盘

1. 在登录界面点击重置密码
2. 插入密码重置 U 盘
3. 按向导重置密码

### 方法二：使用管理员账户

1. 使用其他管理员账户登录
2. 计算机管理 → 本地用户和组 → 用户
3. 右键目标用户 → 设置密码

### 方法三：PE 系统重置

1. 制作 PE 启动 U 盘
2. 从 U 盘启动进入 PE
3. 使用密码清除工具

## 微软账户密码重置

1. 访问 account.live.com/password/reset
2. 输入微软账户邮箱
3. 选择验证方式（邮箱/手机）
4. 设置新密码

## 注意事项

- 重置密码后，EFS 加密的文件将无法访问
- 建议定期创建密码重置盘
- 重要数据做好备份
