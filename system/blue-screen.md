# 蓝屏（BSOD）问题

## 故障现象

电脑突然蓝屏，显示错误代码。

## 常见蓝屏代码

| 代码 | 含义 | 常见原因 |
|------|------|---------|
| CRITICAL_PROCESS_DIED | 关键进程终止 | 系统文件损坏、驱动问题 |
| IRQL_NOT_LESS_OR_EQUAL | 内存访问错误 | 驱动冲突、内存故障 |
| PAGE_FAULT_IN_NONPAGED_AREA | 页面错误 | 内存问题、驱动错误 |
| SYSTEM_THREAD_EXCEPTION_NOT_HANDLED | 系统线程异常 | 驱动问题 |
| DRIVER_IRQL_NOT_LESS_OR_EQUAL | 驱动IRQL错误 | 网卡/显卡驱动问题 |

## 排查步骤

### 1. 记录错误信息

拍照记录蓝屏上的错误代码和失败的操作文件名。

### 2. 进入安全模式

设置 → 系统 → 恢复 → 高级启动 → 疑难解答 → 启动设置 → 启用安全模式

### 3. 分析转储文件

使用 BlueScreenView 工具查看系统转储文件。

### 4. 常见解决方案

**驱动问题：** 更新或回滚驱动，卸载最近安装的驱动

**内存问题：** 运行 Windows 内存诊断，重新插拔内存条

**系统文件损坏：** 运行 sfc /scannow 和 DISM 命令修复

## 预防措施

- 保持驱动和系统更新
- 避免超频
- 使用可靠的杀毒软件
